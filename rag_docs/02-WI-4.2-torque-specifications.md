## WI-4.2 Torque Specifications Reference, Model Y Door Hardware

Standard torque specifications for Model Y door hardware, all stations:

- Door hinge bracket mounting bolts: 45 Nm
- Door hinge pin retaining bolt: 22 Nm
- Door check strap bolt: 18 Nm
- Window regulator mounting screws: 8 Nm

All torque values assume dry, unlubricated threads at room temperature. A re-torque check is required if ambient temperature is below 10°C.

## WI-4.2 Calibration Requirements

Torque wrenches used against this reference must be calibrated within the last 90 days. Check the calibration sticker date before use. Wrenches outside this window must be pulled from service and sent for recalibration.
